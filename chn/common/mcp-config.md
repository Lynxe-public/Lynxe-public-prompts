# MCP服务配置

## 功能说明

MCP（Model Context Protocol）服务允许系统通过外部服务扩展功能。通过配置MCP服务，可以将外部工具和服务集成到系统中，实现更强大的功能。

本系统支持两种连接方式：通过stdio链接或通过SSE/Streaming链接。

## 配置方式

### 通过stdio链接

**第1步** 获取MCP服务信息
- 访问 `https://modelscope.cn/mcp` 获取MCP服务
- 登录账号
- 选择需要的MCP服务，选择"stdio"方式
- 复制配置信息（JSON格式）

**第2步** 进入Lynxe的MCP配置页面
- 点击对话框右上角的配置图标
- 选择"Tools/MCP配置"

**第3步** 导入MCP配置
- 点击"全部导入"按钮
- 将复制的JSON配置信息粘贴到输入框中
- 点击"导入"按钮

**第4步** 等待配置完成
- 等待系统加载配置信息
- 当出现"MCP成功"提示后，配置完成
- MCP服务即可使用

### 通过SSE/Streaming链接

**第1步** 进入MCP配置页面
- 点击对话框右上角的配置图标
- 选择"Tools/MCP配置"
- 点击"新建MCP配置"

**第2步** 获取MCP服务信息
- 登录MCP服务提供平台（如 `mcp.higress.ai`）
- 使用相应账号登录（部分平台提供免费额度）
- 搜索并选择需要的MCP服务
- 打开该服务，复制SSE URL
- **注意**：只复制URL部分，不要包含JSON格式，例如：`https://mcp.higress.ai/mcp-service/xxxxxxxxx/sse`

**第3步** 配置MCP服务
- 回到系统配置页面
- 在配置框中输入以下信息：
  - **MCP名称**：为MCP服务设置一个便于识别的名称
  - **链接类型**：选择 `sse`
  - **URL**：粘贴刚才复制的SSE URL
- 点击"保存"按钮
- 等待系统返回确认信息
- 确认MCP服务显示为"启用状态"

**第4步** 使用MCP服务
- 配置完成后，返回主界面
- 在Func-Agent计划模式中，可以在工具列表中找到新配置的MCP服务
- 选择相应的MCP工具并启用相关功能
- 在计划中使用MCP工具完成相应任务
